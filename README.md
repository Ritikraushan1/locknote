# 🔐 NoteLock

**Simple. Private. Secure.**

NoteLock is a lightweight, distraction-free notes application built for people who value their privacy as much as their ideas. It's a place where your thoughts can live securely without ever leaving your device.

---

## 🍃 The Philosophy

I built NoteLock because I was tired of note-taking apps that required an account to store five words, tracked my every move, and felt bloated with features I didn't need. 

NoteLock is different. It’s built on a few core beliefs:
- **Your data is your business.** No cloud sync, no tracking, no logins.
- **Speed is a feature.** Open, write, save. No friction.
- **Privacy shouldn't be a luxury.** Everything is stored locally on your device, and we're moving towards full database encryption.

## ✨ Features

- **🚀 100% Offline**: Your notes stay on your device. Period.
- **🛡️ Secure by Design**: Built-in privacy screens and no data collection.
- **🏷️ Smart Tagging**: A scalable, search-as-you-type tagging system that suggests matches as you write.
- **📌 Pinned Notes**: Keep your most important thoughts at the top.
- **🗑️ Note Bin**: Accidentally deleted something? Restore it from the Note Bin within 30 days.
- **🌓 Theme Aware**: Looks stunning in both Light and Dark modes.
- **🔍 Fast Search**: Instant searching across titles, contents, and tags.

## 🛠️ Built With

- **React Native** (v0.85)
- **Zustand** (State Management)
- **SQLite** (Local Database)
- **FlashList** (High-performance lists)
- **Day.js** (Date formatting)

## 🏗️ Getting Started

To run NoteLock locally:

1. **Clone the repo**
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run on Android**
   ```bash
   npx react-native run-android
   ```
4. **Run on iOS** (Mac only)
   ```bash
   cd ios && pod install && cd ..
   npx react-native run-ios
   ```

## 🔒 Privacy First

NoteLock does not collect any data. No telemetry, no usage statistics, no crash reports are sent to any server. What you write stays on your device. We recommend regularly backing up your device to ensure you don't lose your notes if your phone is lost or damaged.
